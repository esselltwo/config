Utility scripts:
* `util-scripts/rofi-menu.sh` opens a rofi launcher for executable files under `util-scripts/`.
* Submenus are supported by creating a sibling directory named either `scriptname.d/` or `scriptbase.d/` next to an executable.
* Selecting a script that has a submenu opens the submenu first; the first entry in that submenu runs the base script directly.
